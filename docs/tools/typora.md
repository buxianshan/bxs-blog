## 默认快捷键

- 回到文档顶部：`Ctrl + Home`
- 回到文档底部：`Ctrl + End`

## 自定义快捷键

参考：[https://support.typora.io/Shortcut-Keys/](https://support.typora.io/Shortcut-Keys/)

为了方便切换显示`文件树`和`大纲`，我修改了这两个快捷键（默认的快捷键`Ctrl + Shift + number`不方便按）

- 文件树：`Ctrl + Q`
- 大纲：`Ctrl + W`
