                常用工具

                记录一些自己工作和日常生活中常用的工具软件。

                特别推荐JetBrains系列，非常好用！

                • Python：PyCharm

                • Go：Goland

                • 前端：WebStorm

                • C++/C#：Visual Studio

                # 文本编辑器

                • Notepad ++
                • VS Code

                # 浏览器

                自从用过chrome，再也不想用其它浏览器了。

                # SSH工具

                个人感觉还是 Xshell 最好用（只支持windows）。

                用过 Final Shell、Putty、Mobaxterm。
